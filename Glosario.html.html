Mi glosario personal de Sistemas Operativos

Por: Camila Molina Quintero



A continuacion encontraras las definiciones principales de el glosario



IP

Significa Protocolo de internet



smtp

Significa protocolo simple de transferencia de correo



Sistema Operativo (SO)

Un software que gestiona los recursos de hardware y proporciona servicios comunes para los programas de aplicación.



kernel

La parte central del sistema operativo que maneja los recursos del hardware y proporciona interfaces para los programas de aplicación.



interfaz de usuario (UI)

La parte del sistema operativo que permite la interacción entre el usuario y el sistema, puede ser gráfica (GUI) o de línea de comandos (CLI).



multiusuario

Capacidad de un sistema operativo para permitir que múltiples usuarios accedan al sistema simultáneamente.



multitarea

Capacidad de un sistema operativo para ejecutar múltiples procesos simultáneamente.



multiprocesamiento

La capacidad de un sistema operativo para aprovechar múltiples procesadores o núcleos de CPU



multihilos

Capacidad de un sistema operativo para dividir un proceso en subprocesos que pueden ejecutarse simultáneamente.



multiramas

Capacidad de un sistema operativo para ejecutar múltiples procesos de manera simultánea.



monousuario

Un sistema operativo que permite que solo un usuario acceda al sistema a la vez.



monotarea

Un sistema operativo que solo puede ejecutar un proceso a la vez.



plataforma cruzada

Un sistema operativo que puede ejecutarse en diferentes arquitecturas de hardware.



sistema de archivos

La estructura que el sistema operativo utiliza para organizar y almacenar archivos en un dispositivo de almacenamiento.



administrador de memoria

Parte del sistema operativo que gestiona el acceso a la memoria del sistema y asigna recursos de memoria a los programas.



Administrador de Procesos

Parte del sistema operativo que gestiona la ejecución de programas y asigna recursos del procesador a los procesos.



Administrador de Dispositivos

Parte del sistema operativo que controla la comunicación entre el software y los dispositivos de hardware.



administrador de redes

Parte del sistema operativo que gestiona las conexiones de red y la comunicación entre dispositivos en una red.



virtualización

permite que la computadora pueda compartir sus recursos de hardware



disco duro

dispositivo de almacenamiento de datos para almacenar informacion



dns

permite a los usuarios conectarse a sitios webs



nodo

punto de union de varios elementos



ethernet

protocolo de red para conectar dispositivos y crear redes



sistema de tiempo real

sistema informatico que interactua con su entorno



fibra optica

hilos de transmision de luz



lan

red de computadores que permite la comunicacion y el intercambio de datosS
